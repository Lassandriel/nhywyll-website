Welcome to the official source code for **Nhywyll's Digital Home**. This project is a high-performance, **Cyber-Anime themed hub** designed for VTubers and Content Creators, blending cutting-edge aesthetics with technical excellence.

---

## 🚀 Key Features

- **🎨 Visual & UX Excellence**: Glassmorphism UI, neon glow accents, and 100/100 Lighthouse performance.
- **🌍 Global Presence**: Multilingual (English/German) with persistent states.
- **🔒 Privacy First**: GDPR-compliant cookie management and secure external link attributes.
- **✨ Micro-Interactions**: Smooth transitions, bird-track click effects, and feather particles.
